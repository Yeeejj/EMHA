INSIDE-OUT

Emotion Recognition through Handwriting & Drawing Analysis using CNN-HMM Hybrid Approach

Cyrel Jane A. Edano • University of San Carlos, DCISM • May 2025

1

Assessment Design

3-Part Instrument

  • Part 1: 24-item Self-Report (Likert 1-5) — mix of positive & negative items
  • Part 2: 4 Drawing Exercises — Circles, Dots, Person in Rain, House & Tree
  • Part 3: Writing Exercises — 5 words × 3 styles (Left/Right/Uppercase) + 5 cursive sentences
Participants: 300+ | Ages 18-25 | Right-handed | Single sitting | No erasing
2

Data Collection

Participant Registration

  • Barcode ID assignment (P001 → P300+)
  • Consent form + demographics (age, gender, handedness)
  • Validated against participants.csv to prevent ID collision

Sample Acquisition

  • Handwriting & drawing pages scanned to PNG
  • Organized: DATA/RAW/participant_XXX/
3

Scoring & Labeling

Questionnaire Scoring

  • DASS-21 Depression subscale (0-42)
  • Oxford/SDHS Happiness scale

Emotion Labels

HAPPY: happiness ≥ 40 AND dass < 14
SAD: happiness < 30 OR dass ≥ 14
NEUTRAL: excluded from training
Output: DATA/LABELED/{HAPPY,SAD}/
4

Image Preprocessing

5-Step Pipeline

  • 1. Grayscale conversion (BGR → Gray)
  • 2. Binarization (Otsu's adaptive threshold)
  • 3. Noise removal (morphological open + close)
  • 4. Skew correction (min-area-rect rotation)
  • 5. Normalize & resize to 224×224
Output: DATA/PROCESSED/{HAPPY,SAD}/ — clean 224×224 grayscale PNGs
5

CNN Feature Extraction

Architecture (4 Conv Blocks)

  • Conv2d(1→32) + BatchNorm + ReLU + MaxPool
  • Conv2d(32→64) + BatchNorm + ReLU + MaxPool
  • Conv2d(64→128) + BatchNorm + ReLU + MaxPool
  • Conv2d(128→256) + BatchNorm + ReLU

Dual Output

  • Global: AdaptiveAvgPool → FC(256) → single vector
  • Sequence: Height-average → (batch, 28, 256) left-to-right spatial features
Pretrained via classification head (CrossEntropy + early stopping)
6

HMM Classification

Per-Class Gaussian HMMs

  • HMM_HAPPY: 4 states, diagonal covariance, trained on happy sequences
  • HMM_SAD: 4 states, diagonal covariance, trained on sad sequences

Sequence Classification

  • 28-timestep spatial sequences model left-to-right handwriting progression
  • Compare per-class log-likelihoods → softmax → prediction + confidence
7

Cross-Validation

Stratified 5-Fold CV

  • Balanced class distribution in each fold
  • Per fold:
  • 1. Train CNN with augmentation (rotation, affine, jitter)
  • 2. Extract spatial sequence features
  • 3. Train HMM on extracted sequences
  • 4. Evaluate on held-out fold
8

Evaluation

Metrics

  • Accuracy (target: 80-85%)
  • Precision (macro-averaged)
  • Recall (macro-averaged)
  • F1-Score (macro-averaged, primary metric)
  • Confusion Matrix (HAPPY vs SAD)
Results: mean ± std across 5 folds
9

Final Model & Deployment

Saved Artifacts

  • cnn_model.pth — trained CNN weights
  • hmm_models.pkl — per-class HMM parameters
  • cv_results.json — cross-validation metrics

Inference Pipeline

New Image → Preprocess → CNN Sequences → HMM Score → HAPPY / SAD + Confidence

End-to-End Data Flow

Assessment → Scan → Score & Label → Preprocess → CNN Features → HMM Classify → 5-Fold CV → Evaluate → HAPPY / SAD
