## Key Theories Referenced

### 1. Basic Emotions Theory (Ekman, 1992, 1999)
- Six universal emotions: happiness, sadness, fear, anger, surprise, disgust
- Cross-cultural recognition of facial expressions
- Foundation for the happiness/sadness classification in this thesis

### 2. Graphology Principles
- Handwriting reflects personality and emotional states
- Key features: slant, spacing, pressure, size, baseline, zones

### 3. CNN-HMM Hybrid Architecture
- CNN for spatial feature extraction from handwriting images
- HMM for temporal sequence classification
- Combines strengths of both approaches for improved accuracy

### 4. EMOTHAW Framework
- Novel database for emotional state recognition
- Uses DASS (Depression-Anxiety-Stress Scales) for emotion labeling
- Correlates handwriting features with emotional states

### 5. TEmoDec Framework (Bhattacharya et al., 2022)
- Uses Agglomerative Hierarchical Clustering
- Groups image pixels to identify emotional states
- Tested for 5 emotions with >75% accuracy

#### 3. Plamondon (1989) — Handwriting Motor Models

**APA Citation:**

**Key Findings:**
This paper evaluates multiple motor models for handwriting generation, establishing that second- and third-order systems yield superior simulation results. It introduces the concept that complex movements can be segmented into basic stroke units, with parameters reflecting dynamic processes highly related to handwriting geometry.

**Relevance:**
Provides theoretical foundation for the thesis's stroke-based feature analysis and understanding of handwriting as a motor-control process reflecting internal states.
